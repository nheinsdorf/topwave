The topwave library is a Python 3 package to quickly set up single-particle models and calculate their topological invariants.

Features
--------

- Linear spinwave
- pymatgen interface
- topological invariants
- tightbinding
- supercells
- twisted 2D structures

Roadmap
-------

- Multi Orbital
- Higher order invariants
- BdG Hamiltonians
- Mean Field
